Storia

An Interactive Storybook
Built with Django

Web App for children to learn how to read with interactive storybooks. It turns plain stories into interactive stories with an analyses of the child's progress. Over 16 classes created to build the database (SQLite) and HTML-Templating for the Front-end interactive presentation. Uses a REST Framework to capture user actions then store them in the database.

A Sample of a
UX-Designed Website

I studied "User Story Templates" to devolop a plan to update the website. Responsive design was added by using Media Queries, Semantic HTML, CSS Grid and Flexbox. For a better Screen Reader experience, added alt tags for images, aria-role attributes, aria-labels and aria-labelled-by.

TEMPCONVERT

Interactive App Built with React 16

TempConVert is a web app that allows the user to convert Kelvin to Celsius and Fahrenheit. This web app accepts input (K) and then posts a response based on the conversion to Fahrenheit and Celsius. The goal of this project was to transform this console-based app into a web app using React.

Ravenous

An Interactive App Built with React 16 & the Yelp API

An interactive app using the Yelp API to find places to eat. User can input a type of food and location for the search criteria then sort by Best Match, Highest Rated and Most Reviewed. Built using Stateful and Stateless components, passing props, mapping data and then rendering. Use of Fetch() for the API calls.

Data Analysis

Gun Deaths in the US: Exploring Data Using Python

This project was done in Jupyter Notebook using a dataset from FiveThirtyEight (2012 to 2014) and data from the Census. Used CSV and Datetime modules to help sort the data. Explored the data by gender, race, and amount of deaths per month and year.

Code Challenge: Factorialize a Number

Used a 'for' loop with the range function range(start, stop[,step]) to generate a range of numbers, starting with 1 and ending with the argument + 1. The second parameter in range() is exclusionary, so num + 1 is used to include the argument. Then multiplied each number to the next number.
